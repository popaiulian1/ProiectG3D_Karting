# ProiectG3D_Karting

Small project for university in which we've created a karting track with a kart that you can drive around.
# Controls:
## General Camera Movement:
"R" -> reset
"E" -> lock to kart seat

## Kart Movement and Controls: 
### Movement:
"W" -> accelerate
"S" -> brake
"A" -> turn left
"D" -> turn right
"SPACE" -> reverse

### Controls:
"Q" -> reset kart to start/finish line position
